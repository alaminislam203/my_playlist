# my_playlist
🎧 My Playlist

A simple and powerful playlist management system where you can organize, play, and manage your favorite media effortlessly.

🚀 Features
🎵 Create and manage playlists
➕ Add / ❌ Remove songs easily
🔍 Search songs quickly
▶️ Smooth playback experience
💾 Save playlists locally / database
⚡ Lightweight and fast performance
🛠️ Tech Stack
Frontend: (e.g. HTML, CSS, JavaScript / React)
Backend: (e.g. Node.js / C# / Django)
Database: (e.g. MongoDB / MySQL / Local Storage)
Media Engine: (if used, e.g. VLC plugin / HTML5 Audio)
